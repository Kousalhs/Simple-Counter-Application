# Simple Counter Application

A simple passenger counter application built using HTML, CSS and JavaScript. This project was created while learning core frontend fundamentals.

---

## Built with

- HTML
- CSS
- Javascript (Vanilla)
- Netlify

---

## Features

- Counter interface
- Increment and save logic using JavaScript
- Live display of current count
- Clean and minimal UI
- Lighweight static web page

---

**What I learned in this section:**

- Manipulating the DOM to update page content
- Using JavaScript variables and functions for state
- Connecting button events to behavior
- Handling number logic and text display
- Basic project structure with HTML, CSS & JS
